# PicGo + Gitee

网上推荐七牛云阿里云都是要租赁服务器，太麻烦还要钱，微博现在挂链接又很厉害。大部分人选择用github，但是github虽好却是国外的网站，速度终究比不上国内网站，研究了小半天，终于发现完美的解决方案。

最终方案：PicGo + Gitee 来实现 markdown 图床

# 安装软件

  • PicGo
  • Node.js

先首先安装 PicGo ，打开主界面

然后安装好 Node.js，选择最底下和插件设置，搜索gitee，安装gitee-upploader

# 创建Gitee仓库

点击右上角的+号，新建仓库

新建仓库的要点如下：

  1. 输入一个仓库名称
  2. 其次仓库设为公开
  3. 勾选使用Readme文件初始化这个仓库， 勾选上自动给你的仓库建立master分支

点击创建进入下一步

# 配置PicGo

安装了gitee-uploader插件之后，我们开始配置

  • repo：用户名/仓库名
  • branch：分支，这里写上master
  • token：填gitee的私人令牌
  • path：路径，一般填img

# 获取token

  1. 点击头像，进入设置

  1. 找到左边的安全设置的私人令牌

  1. 点击<生成新令牌>，把project这一项勾上，其他不用勾选

然后提交，注意这串数字只会明文显示一次，记得保存，搞丢了又要重新生成一个。

这串数字就是你的token，复制到配置里面去，然后将它设置为默认图床。

大功告成。
