# 冒泡排序

概念

冒泡选择法的本质在于交换.

每一趟把最值往后放，每结束一趟将减少一次比较。

#include <stdio.h>
int main(void){
    int a[20] = {8, 6, 9, 3, 2, 1, 5, 4, 7, 0};
    int temp, i, j;
    //进行n-1趟
    for (i = 1; i <= 9; i++){ 
        //从a[0]到a[n-i-1]都与下一个数比较
        for (j = 0; j <= 10 - i - 1; j++){ 
            if (a[j] > a[j + 1]){
                temp = a[j];
                a[j] = a[j + 1];
                a[j + 1] = temp;
            }
        }
    }
    for (i = 0; i <= 9; i++){
        printf("a[%d] = %d\n", i, a[i]);
    }
    return 0;
}
输出结果 
a[0] = 0
a[1] = 1
a[2] = 2
a[3] = 3
a[4] = 4
a[5] = 5
a[6] = 6
a[7] = 7
a[8] = 8
a[9] = 9
